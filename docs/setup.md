### Prerequisites
- Node.js (v18 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```
   The server will run on `http://localhost:3000`.

### Testing

To run the test suite:
```bash

```
